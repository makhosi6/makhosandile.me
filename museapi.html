Muse API

Project information

  • Category: REST API,
  • Stack: PHP, Laravel,Docker, Tailwind CSS, Livewire, Fortify and Sanctum, GraphQL
  • Project Code: Link
  • Project URL: https://museapiv1-k5gjw.app

This project was scaffolded with Laravel Jetstream. I prefer Jetstream because if offers a elegent way of authenticating users and user tokens. Jetstream is backed by, Livewire, Fortify and Sanctum. Sanctum allows my users to create tokens with different abilities and permissions.
For the sake of maintaining data integrity external API users can't delete, create or update.
This API was primarily meant to be consumed by the Muse UI app.
This project was dockerized and hosted on Digital Ocean.
